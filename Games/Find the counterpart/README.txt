|||||||||||||||||||||| Directions to use: ||||||||||||||||||||||


Run the "opposite.html" file in your browser and play the game.


|||||||||||||||||||||| Information |||||||||||||||||||||||||

User can set the number of choices to be shown. Follow the below steps:

1. Go to "hindi.json" file in "Find the counterpart" folder i.e., the folder in which the main file is present.
2. Find the parameter "num_of_choices" and set the desired number. NOTE : the number against this parameter should be a sane figure. :)
